# 🌦 Weather App - Task 3

## 📌 Project Description
This Weather App fetches real-time weather data using the OpenWeather API and displays weather information for any city entered by the user.

---

## 🚀 Features
- Search weather by city name
- Real-time temperature display
- Humidity information
- Wind speed details
- Weather icons
- Error handling for invalid city names
- Loading message while fetching data
- Responsive UI design

---

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeather API
